‚ùì FAQ

Q: What file formats are supported? A: Supports ZIP and folders containing images.
Q: What if WiFi Transfer won't connect? A: 1) Ensure phone and computer are connected to same WiFi; 2) Check if firewall is blocking; 3) Try restarting WiFi Transfer service.
Q: Is reading progress automatically saved? A: Yes, the app automatically saves the current page every time you turn pages or exit the reader. Next time you open it will jump directly to the last reading position.
Q: How to batch import books? A: Tap "Add" button on the bookshelf interface, select "Batch Add Books", then select a folder containing multiple comic files.
Q: What image formats are supported? A: Supports common image formats including JPEG, PNG, WebP, etc. You can pack images into ZIP files or directly import folders containing images.

Tips & Tricks

File Import

  • WiFi Transfer: Upload files through browser, suitable for transferring from computer to mobile devices
  • Batch Import: Select a folder containing multiple comic files and import at once
  • File Naming: Using clear file names helps automatically recognize series and volume numbers

Book Management

  • Tag Categories: Add tags to books for easy filtering and searching later
  • Reading Progress: The app automatically records reading position and continues reading when you open it next time
