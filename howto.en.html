Method 1: Local File Import

  1. Tap the "Import" button in the top-right corner
  2. Select "Add Books" for single book, or "Batch Add Books" for multiple books
  3. Choose the book folder and wait for scanning to complete

Method 2: Network Share Import (For NAS Users)

  1. Mount network share in Files app
    • Open the system "Files" app
    • Tap "‚Ä¶" in top-right ‚Üí "Connect to Server"
    • Enter address: smb://server-IP/shared-folder
    • Enter username and password
  2. Import from network location
    • Tap "Import" in Manga Grove
    • Select the mounted network location in file picker
    • Choose the book folders to import
‚ö†Ô∏è Note: Files will be copied to local storage, ensure sufficient space

Method 3: WiFi Transfer (Recommended)

The most convenient way, supports transferring multiple books at once!

  1. Start service
    • Tap "Import" ‚Üí "WiFi Transfer" on bookshelf
    • Tap "Start WiFi Transfer"
    • Note the displayed address (e.g., http://192.168.1.100:8080)
  2. Upload from computer
    • Ensure computer and phone/iPad are on the same WiFi
    • Open the displayed address in computer browser
    • Select files or drag and drop to upload
  3. Batch upload support
    • Upload multiple ZIP archives at once
    • Each ZIP will be automatically extracted and imported

Supported Formats

Image Formats: JPG, PNG, GIF, WebP

Container Formats:

  • Folders (recommended)
  • ZIP/CBZ archives (WiFi Transfer only)

Not Supported: RAR, 7z, PDF

File Naming Recommendations

Strongly recommend using numeric names for images:

  • Correct cover selection (avoids advertisement images)
  • Pages in correct order

‚ùìFAQ

Import Related

Q: Can't find imported books?

  • Ensure folder contains supported image formats
  • Each book must be in a separate folder
  • Image files should be directly in folder root

Q: How to import RAR format comics?

  • Extract with decompression software first
  • Then import the extracted folder

Q: Why is the cover showing an advertisement?

  • Rename images to pure numbers: 001.jpg, 002.jpg...
  • App will automatically select numerically named images as cover

WiFi Transfer Related

Q: Can't connect to WiFi Transfer?

  • Ensure phone and computer are on the same WiFi
  • Check computer firewall settings
  • Enter complete address including http:// and port number

Q: Upload failed?

  • Check if network connection is stable
  • Confirm device has sufficient storage
  • Try restarting WiFi Transfer service

Network Share Related

Q: Can't access network location?

  • Ensure stable network during import
  • Recommend using WiFi Transfer as alternative
  • Or download files locally first then import
